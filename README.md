# Text-to-Speech API

A FastAPI-based backend service that converts text to speech using the Google Text-to-Speech (`gtts`) library. This API allows you to generate spoken audio from text input with various customization options.

## Features
- Convert text to speech in multiple languages.
- Configure speech speed and volume.
- Easy-to-use API with automatic documentation.
